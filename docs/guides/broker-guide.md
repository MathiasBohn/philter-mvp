## Pre-Submission QA Process

### Your Role as Broker

**Before Phase 2:**
- Basic review of application completeness
- Help applicant with questions

**With Phase 2:**
- **Comprehensive QA** before submission
- **Verify data accuracy** and completeness
- **Catch red flags** early
- **Optimize application** for faster approval

**Your value:** Clean, complete applications move faster and have higher approval rates

---

Use this checklist before submitting any application:

#### 1. Navigation Sidebar Check
- [ ] Open application

- [ ] No sections show ðŸ”´ red exclamation (needs attention)

#### 2. Personal Information
- [ ] Full legal name (matches ID)
- [ ] Current address complete

#### 3. Employment Section (Enhanced)
- [ ] At least one CURRENT employment record
- [ ] Each employment record complete:
  - [ ] Employer name
  - [ ] Position/title
  - [ ] Start date (end date if not current)
  - [ ] Annual income
  - [ ] **Supervisor name, phone, email** â­ NEW

  - [ ] Business name
  - [ ] Tax ID/EIN
  - [ ] Business type
  - [ ] Years in business

- Income significantly above/below norm for position

#### 4. Financial Information (Enhanced)
- [ ] **Assets** entered in detail:
  - [ ] Cash & Checking
  - [ ] Savings
  - [ ] Stocks/Bonds
  - [ ] Retirement accounts
  - [ ] Other categories as applicable
- [ ] **Liabilities** entered in detail:
  - [ ] Credit card debt
  - [ ] Auto loans
  - [ ] Student loans
  - [ ] Other debts
- [ ] **Supporting documents** uploaded:
  - [ ] Bank statements (2-3 months)
  - [ ] Investment/brokerage statements
  - [ ] Pay stubs or tax returns
- [ ] **Financial ratios** look good:
  - [ ] Debt-to-Income (DTI) < 40% (ideally < 30%)
  - [ ] Liquid assets > 2 years rent
  - [ ] Net worth positive and substantial

**Red flags:**
- DTI > 40%
- Liquid assets < 1 year rent
- Large recent cash deposits (anti-money laundering concern)
- Unexplained large debts

- [ ] All properties disclosed
- [ ] Each property has:
  - [ ] Complete address
  - [ ] Property type (Primary, Investment, Vacation)
  - [ ] Purchase price and date
  - [ ] Current market value (realistic)
  - [ ] Mortgage balance
  - [ ] Monthly payment
- [ ] For investment properties:
  - [ ] Rental income documented
- [ ] Total equity calculates correctly
- [ ] Property values seem realistic (compare to Zillow)

**Pro Tip:** Real estate ownership demonstrates financial sophistication. If your client owns investment properties, ensure they're prominently displayed.

---

#### 6. Documents
- [ ] All required documents uploaded
- [ ] Documents are legible (not blurry photos)
- [ ] Documents are current (within 60-90 days for financials)
- [ ] File names descriptive (not "IMG_1234.jpg")
- [ ] Correct document types:
  - PDFs for official documents
  - High-quality JPG/PNG for ID scans

**Required documents (typical):**
- Government-issued ID
- Proof of income (pay stubs or tax returns)
- Bank statements (2-3 months)
- Credit report (if pre-pulled)
- Reference letters (if available)

---

#### 7. References
- [ ] 2-3 references provided
- [ ] Include current/former landlords
- [ ] Complete contact info (name, phone, email, relationship)
- [ ] References are reachable (not old/disconnected numbers)

**Pro Tip:** Call references yourself before submission to verify they'll respond positively

---

#### 8. Acknowledgements (Enhanced - 8 Forms)
- [ ] All acknowledgements completed
- [ ] Electronic signatures captured
- [ ] Timestamps recorded
- [ ] **Pet declaration** (if applicable):
  - [ ] Pet details complete
  - [ ] Pet complies with building policy
  - [ ] Pet deposit/fee calculated
- [ ] **Move-in date** commitment:
  - [ ] Date is realistic (30+ days)
  - [ ] Requirements acknowledged

**Critical:** Applications cannot be submitted without all acknowledgements. Verify before final review.

---

#### 9. Building Policies
- [ ] Applicant acknowledged: "I have read and understand building policies"
- [ ] Timestamp recorded
- [ ] If applicant has questions about policies, address before submission

---

#### 10. Cover Letter (New)
- [ ] Cover letter written
- [ ] Professional tone
- [ ] Addresses why applicant wants to live in building
- [ ] Mentions financial stability/responsibility
- [ ] No red flags (complaints, negativity, inconsistencies)
- [ ] Length appropriate (not too short, not rambling)

**Pro Tip:** Review cover letter carefully. A strong, personal letter can make the difference for borderline applications.

## Reviewing Enhanced Sections

### Employment Review Tips

**Navigate to:** Employment section

**What to review:**
1. **Current employment verified:**
   - Employer is legitimate (Google search if unfamiliar)
   - Position matches LinkedIn profile (if available)
   - Income is reasonable for position/location
   - Supervisor contact is complete

2. **Employment history makes sense:**
   - No unexplained gaps
   - Career progression logical
   - Job changes have reasons (if many)

3. **Self-employment (if applicable):**
   - Business name matches tax documents
   - Tax ID format valid
   - Documentation adequate (Schedule C, 1099s, business bank statements)

**How to help applicant:**
- If supervisor info missing: "Admin will need to verify employment. Please provide your supervisor's contact information."
- If self-employed: "Please upload your last 2 years tax returns (Schedule C), 1099s if applicable, and business bank statements"

---

### Financial Review Tips

**Navigate to:** Financial Information

**What to review:**

1. **Assets look comprehensive:**
   - All accounts listed (not just one)
   - Values reasonable
   - Supporting statements uploaded
   - No round numbers (suspicious - may be estimated)

2. **Liabilities disclosed:**
   - Credit cards listed if applicant has any
   - Auto loans if applicant owns car
   - Student loans if applicant went to college
   - Compare to credit report (if available)

3. **Calculations make sense:**
   - **DTI:** Should be < 40%. Calculate: (Total Monthly Debt / Monthly Income) Ã— 100%
   - **Liquid Assets:** Should cover 1-2+ years rent
   - **Net Worth:** Should be positive and substantial

**Red flags:**
- All assets in cash (money laundering concern)
- Assets + income don't match lifestyle (luxury address, expensive car)
- Large recent deposits without explanation
- Liabilities much higher than expected for age/income

**How to help applicant:**
- "Your DTI is 45%, which may concern the board. Can you pay down credit card debt before submission?"
- "Your liquid assets are low. Consider showing additional accounts or investments."
- "Large deposit in savings - please provide letter of explanation (gift, bonus, sale of asset?)"

---

### Real Estate Review Tips

**Navigate to:** Real Estate Holdings

**What to review:**
1. **Property values realistic:**
   - Check Zillow/recent sales for approximate value
   - Inflated values are red flag

2. **Equity positive:**
   - Current Value > Mortgage Balance
   - Negative equity (underwater) is concern

3. **Rental income documented:**
   - If investment property, should show rental income
   - Upload lease agreement or rent roll

**How to help applicant:**
- "Your property value seems high. Zillow shows $X. Can you provide recent appraisal?"
- "Please upload your rental lease to document rental income of $X/month"

---

### Acknowledgements Review Tips

**Navigate to:** Acknowledgements

**What to verify:**
1. All 8 checkboxes checked (or 7 if no pets)
2. Electronic signatures captured (applicant typed name)
3. Timestamps present
4. Pet details complete (if applicable)

**If pet declared:**
- Verify pet complies with building policy:
  - Type allowed (dog, cat, etc.)
  - Breed allowed (no restricted breeds)
  - Size within limits (if applicable)
  - Weight within limits
- Service/emotional support animal documented
- Pet deposit calculated correctly

**How to help applicant:**
- "Building allows dogs up to 40 lbs. Your dog is 55 lbs - this may be an issue. Contact building to request exception."
- "You claimed service animal. Please upload certification/documentation."

---

### Cover Letter Review Tips

**Navigate to:** Personal Introduction

**What to review:**
1. **Content quality:**
   - Introduces applicant and family
   - Explains why they want to live in building
   - Demonstrates financial responsibility
   - Professional tone
   - Good grammar/spelling

2. **Red flags:**
   - Mentions problems (legal, financial, health)
   - Negative tone (complaints about current building)
   - Inconsistent with application data
   - Too short (< 200 chars) or generic
   - Too long/rambling (> 2000 chars)

3. **Cultural fit:**
   - Shows community values
   - Mentions amenities/features
   - Respectful of building rules

**How to help applicant:**
- "Your cover letter is very brief. Consider expanding to 300-500 words to make a stronger impression."
- "You mentioned financial difficulties in 2020. This may concern the board. Consider reframing positively: 'Despite 2020 challenges, I've maintained strong finances...'"
- "Your letter is generic. Mention specific features of [Building Name] that attract you."

### Handling Common Questions

**Q: "Why do they need so much financial information?"**
A: "Buildings want to ensure residents can afford rent/mortgage long-term. Detailed financials show your stability and reduce follow-up questions, speeding up your approval."

**Q: "Do I really need to list all my assets?"**
A: "Yes - the more assets you show, the stronger your application. Think of it as showing off your financial strength!"

**Q: "What's a good cover letter?"**
A: "Introduce yourself, explain why you love this building, and show you're financially responsible and a good neighbor. Be genuine and professional. Would you like to see an example?"

**Q: "I don't own real estate - should I leave that section blank?"**
A: "Yes, if you don't own any real estate, you can leave that section empty. No problem - many renters don't own property."

**Q: "Can I submit without the cover letter?"**
A: "Technically yes, but I strongly recommend it. It's your chance to make a personal connection with the board. A good letter can make the difference for borderline applications."

## Best Practices

### Maximize Approval Rates

**1. Submit Complete Applications**
- Use QA checklist every time
- Verify all green checkmarks in sidebar
- Check quality score >75

**2. Optimize Financial Presentation**
- Show ALL assets (don't hide accounts)
- Pay down high credit card balances before submission
- Explain large deposits or unusual items
- Ensure DTI < 40% (ideally < 30%)

**3. Strong Cover Letter**
- Help applicant write compelling, genuine letter
- Mention building-specific features
- Show community values
- Professional but personal tone

**4. Quality Documents**
- Scan (don't photograph) official documents
- Ensure legibility
- Current documents (< 60 days for financials)
- Organized and labeled

**5. Proactive RFI Management**
- If you anticipate admin will ask for something, include it upfront
- Add notes to explain unusual items
- Be thorough

### Time Management

**Efficient workflow:**
1. **Batch review:** Set aside time weekly to QA all pending applications
2. **Use dashboard filters:** "Ready for submission" view
3. **Prioritize:** Oldest > Newest
4. **Set applicant deadlines:** "Please complete by Friday so I can review and submit Monday"
5. **Track in CRM:** Note application status and follow-up dates

**Average time per application:**
- Thorough QA review: 20-30 minutes
- Feedback to applicant: 10 minutes
- Final submission check: 5 minutes
- **Total:** ~45 minutes per application

**ROI:** 45 minutes of QA = fewer RFIs, faster approvals, happier clients, better reviews

## Common Scenarios

### Scenario 1: Self-Employed Applicant

**Challenge:** Self-employment income harder to verify than W-2 employment

**Your approach:**
1. **Extra documentation:**
   - Last 2 years tax returns (full return, not just 1040)
   - Schedule C (business income/expenses)
   - 1099s received
   - Business bank statements (3-6 months)
   - CPA letter verifying income
   - Business license/registration

2. **Cover letter:**
   - Help applicant explain business (what they do, how long, stability)
   - Emphasize consistent income and growth

3. **Financial strength:**
   - Self-employed applicants should show higher reserves (3+ years rent recommended)
   - Demonstrate stability (same business 3+ years ideal)

**Pro Tip:** Self-employed applicants are often high net worth. Emphasize assets and business success in presentation.

---

### Scenario 2: Recent Graduate / First-Time Renter

**Challenge:** Limited employment history, limited assets

**Your approach:**
1. **Guarantor consideration:**
   - Many buildings accept parental guarantor
   - Guarantor provides financial backing
   - Include guarantor financials in application

2. **Emphasize strengths:**
   - New job with reputable company
   - Strong starting salary
   - Excellent credit (if applicable)
   - Educational achievement

3. **Cover letter:**
   - Professional career start
   - Commitment to building
   - Responsible tenant history (if any)

**Pro Tip:** Even if not required, suggest guarantor proactively for marginal applications

---

### Scenario 3: High Net Worth, Complex Finances

**Challenge:** Multiple business entities, trusts, investments - complicated to present

**Your approach:**
1. **Simplify presentation:**
   - Create summary sheet (one-pager with all accounts totaled)
   - Organize documents by category (Personal, Business, Trusts, Investments)
   - Add explanatory notes for complex structures

2. **CPA letter:**
   - Request CPA summary letter certifying net worth and income
   - Adds credibility
   - Simplifies admin review

3. **Cover letter:**
   - Don't over-explain complexity
   - Focus on stability and financial strength
   - Mention professional management (CPA, wealth advisor)

---

### Scenario 4: Application with Issues

**Challenge:** Applicant has past credit issues, employment gap, or other concerns

**Your approach:**
1. **Disclose proactively:**
   - Don't hide issues (admin will find them)
   - Include letter of explanation with application
   - Show how issue is resolved/past

2. **Demonstrate recovery:**
   - Credit: Show recent payment history, increasing score
   - Employment: Show current stable employment
   - Financial: Show rebuilding (growing assets, decreasing debts)

3. **Emphasize positives:**
   - Current strong position
   - Lessons learned
   - Responsibility and recovery

**Pro Tip:** Applicants who proactively explain issues and show recovery often get approved. It's the hiding of issues that causes denials.

---

### Scenario 5: Tight Timeline

**Challenge:** Applicant needs approval in < 2 weeks

**Your approach:**
1. **Set realistic expectations:**
   - Explain typical review timelines
   - Stress importance of complete application
   - Board review timelines vary

2. **Intensive QA:**
   - Work with applicant daily to complete sections
   - Review in real-time as they fill out
   - Submit as soon as complete (don't wait)

3. **Communicate with management:**
   - After submission, send note: "This applicant has tight timeline due to [reason]. Please let me know if any issues immediately."
   - Monitor daily
   - Respond to RFIs within hours

**Pro Tip:** Your responsiveness directly impacts timeline. Be available and proactive.

## FAQs

**Q: How long does it take applicants to complete the new application?**
A: 45-60 minutes if they have all documents ready. Gathering documents can take longer.

**Q: Do all 11 sections need to be completed?**
A: Core sections yes. Real Estate Holdings is only needed if applicant owns property. Documents section depends on building requirements.

**Q: Can I fill out the application for my client?**
A: You can help, but applicant must review and sign all acknowledgements personally. Electronic signature must be theirs.

**Q: What if applicant doesn't have a cover letter?**
A: Encouraged but not always required (depends on building). I recommend always including one.

**Q: What's a good quality score?**
A: 90+ is excellent, 75-89 is good, 60-74 is acceptable. Don't submit below 60 without addressing issues.

**Q: How do I know if application is ready to submit?**
A: All sections green checkmark, quality score >75, you've reviewed each section personally, no red flags.

**Q: What if admin sends RFI after submission?**
A: Notify applicant immediately, get response, upload to application. Fast response reduces delays.

**Q: Can I see application after submission?**
A: Yes - you have read-only access and can see admin notes and status updates.
